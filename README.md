This project is an adaptive chess bot that changes its "difficulty" or "ELO" based on how well the player is performing against it.
It was written using Python and uses the Komodo Dragon chess engine for handling everything related to engine decision making.

# You will need to source a copy of LLaMa 3.2:3b or otherwise in order to access feedback. Any other models will work but you will need to edit and recompile.
